How Connected is the World?



Background Information

The following graphs present data regarding to the number of internet users and the number of cellular and broadband subscriptions. The aim is to visualise how the amount of people using internet changes over time for different countries.

Data Trends

From the heatmap we can see that the percentage of people using internet increases over time. Comparing the different countries in the world map we can see that some areas have less internet users then others. For example most countries in Africa have less people using internet compared to others. We can also see this trend in south Asia and middle eastern countries From the bar chart and bubble plot we can see that cellular subscriptions and broadband subscriptions also increases over time but unlike internet usage there hasn't been much change since 2010. Since the plots only show the top 10 countries with the most internet users we can expect that this is not the case for all countries. From the bubble chart we are also able to see that cellular subscriptions are more common then broadband subscriptions but since some people use multiple cellular subscriptions, it is not a fair comparision between the two.
